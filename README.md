## Використане API

Для роботи додатка використовується **Firebase Realtime Database**, для доступа до якого використовується масив такого виду:

```js
const firebaseConfig = {
  apiKey: "",
  authDomain: "",
  databaseURL: "",
  projectId: "",
  storageBucket: "",
  messagingSenderId: "",
  appId: "",
};
```

Цей сервіс дозволяє записувати і зчитувати дані з бази даних.
